# Retail Margin Resilience Analytics — Project Scope

## Project Objective

Retail Margin Resilience Analytics is a public-company-informed retail analytics case study designed to analyze how a large retailer can protect profit margins under operational pressure.

The project uses Target Corporation as the public-company reference and combines filing research, simulated SKU-store operational data, analytics engineering, machine learning, and executive dashboarding.

## Core Business Question

How can a large retailer protect profit while balancing demand volatility, stockouts, overstock, returns, shrink, markdowns, and fulfillment delays?

## Anchor Company

Target Corporation

Target is used as the public-company reference because its recent filings and business disclosures connect directly to the project’s focus areas:

- Gross margin pressure
- Markdown activity
- Inventory shrink
- Category mix effects
- Inventory valuation and markdown reserves
- Omnichannel fulfillment and retail operations

## Project Type

This is not a real internal Target dataset.

This project uses:

1. Public-company filings and annual reports for business context
2. Simulated retail operational data generated with realistic assumptions
3. Analytics engineering models built in PostgreSQL and dbt
4. Machine learning outputs for forecasting and risk scoring
5. Power BI dashboards for executive decision-making

## Included Scope

The first version will not include:

- Real-time streaming
- Live APIs
- Customer-level personalization
- Deep learning
- Reinforcement learning
- Full cloud deployment
- Real Target internal data
